# 🏏 SportGPT — AI Sports Content Generator

**SportGPT ** is a Streamlit-based AI application that automatically generates professional sports articles using live cricket match data and a powerful LLM.
It combines **real-time match insights**, **AI content generation**, and **user authentication** to create a mini AI-powered sports journalism platform.

---

## 🖥️ Demo UI

Main interface includes:

* **AI Content Generator**
* **Trending Matches Sidebar**
* **Article History Page**
* **User Login/Register**

---

## 🏗️ Tech Stack

**Frontend**

* Streamlit

**Backend**

* Python

**AI Model**

* Groq API (Llama 3.3)

**Database**

* SQLite

**APIs**

* CricAPI (Live cricket data)

```

## 📸 Example Usage

1. Login or register
2. Enter a sports topic like:

```
Write an IPL match report for MI vs CSK
```

3. The AI will generate a professional article using live match data.
